A Monkey Island inspired twitterbot

Sat 08 August 2015 | tags: python, twitterbot, twitter

Just for fun, I've made a rather silly twitterbot inspired by Monkey Island's 'insult sword-fighting'. For example, tweet "@fight_guybrush you fight like a dairy farmer" and it will reply "How appropriate! You fight like a cow!" within the hour. Other Monkey Island insults should work too. It uses the python tweepy library, which was a bit of a challenge as the documentation seems to be quite incomplete. I've used cron to schedule the bot script to run every hour. The code is Here: https://github.com/samfrances/threepwood
